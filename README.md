# Python Data Science Basics

This repository documents my learning while working through the **W3Schools Data Science tutorial** to build a foundational understanding of data science using Python.

🔗 Tutorial: https://www.w3schools.com/datascience/default.asp

I began this project with **no prior experience in data science**, using it as a structured way to explore the fundamental concepts used when working with data. The notebooks and examples in this repository follow the general structure of the tutorial and serve as a place to experiment with ideas, document notes, and practice writing small pieces of Python code related to data analysis.

The goal is to better understand how data can be organized, analyzed, and interpreted using Python and common data science techniques.

# Sections

## Introduction to Data Science

The introduction focuses on the basic ideas behind data science and how Python can be used to work with data.

Topics explored in this section include:

* What data is and how it can be represented
* How data is organized in tables
* Using Python to work with structured data
* Working with DataFrames
* Writing simple functions
* Preparing and cleaning data before analysis

This section provides an overview of how raw data is typically **structured and prepared before meaningful analysis can occur**.

---

## Mathematics for Data Science

Mathematics forms the foundation of many data science techniques. This section introduces some of the basic mathematical ideas that are useful when analyzing relationships between variables.

Topics include:

* Linear functions
* Plotting mathematical functions
* Understanding slope and intercept
* Representing relationships between variables visually

These ideas help illustrate how changes in one variable can affect another and how mathematical models can be used to represent those relationships.

---

## Statistics

Statistics provides the tools needed to describe, summarize, and interpret data.

This section introduces statistical concepts commonly used when analyzing datasets, including:

* Basic statistical concepts and terminology
* Percentiles and how values are distributed in a dataset
* Variance and standard deviation as measures of variability
* Correlation between variables
* Correlation matrices for comparing multiple variables
* The difference between correlation and causality

Understanding these concepts helps explain **patterns, variability, and relationships within data**.

---

## Advanced Data Science Concepts

The advanced section introduces techniques used to model and interpret relationships in data more formally.

Topics include:

* Linear regression
* Interpreting regression tables
* Understanding regression coefficients
* Using p-values to evaluate statistical significance
* Understanding R-squared values
* Applying regression models to real examples

These concepts show how mathematical and statistical methods can be used to **build models that describe or predict relationships between variables**.
